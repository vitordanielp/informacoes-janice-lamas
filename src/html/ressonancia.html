MAGNETOM Altea

Especificações técnicas
Peso máximo suportado 150kg
Comprimento do equipamento 1,57m
Força de campo 1,5 Tesla
Tamanho do furo 70cm

Ressonâncias Com Contraste no Mesmo Dia

Permitidas

  • Crânio + tórax
  • Crânio + coluna
  • Crânio + ouvido (ossos temporais, mastóides, etc)
  • Crânio + órbitas (olhos)
  • Crânio + hipófise
  • Crânio + sela túrcica
  • Crânio + perfusão + espectroscopia

Restritas

  • Mamas + qualquer outra
  • Pelve + qualquer outra

Observação: exames sem contraste não têm restrições.
